# General

- The API tester UI should show what the required parameters are based on @docs/05-api-tech-spec.md
- If the middleware doesn't have all the required parameter to work, the middleware should throw an error to help avoid making invalid requests to the API
- A `options` parameter is found in a lot of the endpoints. It's unclear what should or could go in that field
